## Guidance on Secure Handling

1. **Store addresses only in encrypted databases** (e.g., use field-level encryption for address columns).
2. **Restrict access** so only authorized personnel and services can access address data.
3. **Avoid exposing whole addresses in logs or error messages**.
4. **Mask or truncate addresses** where possible in user interfaces unless full detail is absolutely required.
5. **Ensure compliance** with local data regulations (GDPR, Swiss FADP, etc.).
